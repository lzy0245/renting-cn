## First Message To Landlord Or Agent

```text
你好，我想先确认几个签约前信息：
1. 这套是否就是图片/视频里的 exact 房源？现在还能看吗，最早什么时候入住？
2. 你这边的角色是房东、授权人、中介、当前租客转租、室友招租，还是公寓运营方？
3. 总费用包括哪些：租金、押金、付款周期、中介费、服务费、物业、水电燃气、网费、清洁费？
4. 签约方和收款方是谁？签约前能否核验产权/授权/原租约和转租同意？
5. 押金什么情况下会扣，退租后多久退回？提前退租规则是什么？
6. 能否发一个今天拍的完整视频：入户门、客厅/卧室、厨房、卫生间、窗外、楼道和小区入口？
```

## Negotiation Wording

```text
我这边比较认真考虑这套房，但还需要把总成本和风险确认清楚。
如果签约方、收款方和全部费用都能核验清楚，并且把[具体事项]写进合同，我可以尽快推进。
价格方面，附近类似房源我看到大概在[区间]，这套如果能到[目标价/免除某费用/调整付款周期]，我会更倾向直接定。
```
